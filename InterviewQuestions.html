Manual Testing Questions

Q1. Difference between Severity and Priority?

Severity: Impact of the defect on functionality.

Priority: Urgency to fix the defect.

  • Login button not working → High Severity + High Priority
  • Typo in homepage → Low Severity + High Priority

  2. Test Planning
  3. Test Case Development

  5. Test Execution

Q4. Difference between == and === ?

== compares values only.

=== compares value + datatype.

5 == "5"   // true
5 === "5"  // false

Q7. What is Union Type?

let value: string | number;

Variable can store multiple datatypes.

TypeScript Data Types

1. Number

Used for numeric values.

        let age: number = 25;
        let price: number = 99.99;

2. String

Used for text values.

        let name: string = "John";

3. Boolean

Used for true/false values.

        let isLoggedIn: boolean = true;

4. Array

Used to store multiple values.

        let numbers: number[] = [1, 2, 3];
        
        let names: string[] = ["A", "B", "C"];

5. Tuple

Fixed number of elements with fixed types.

        let employee: [number, string] = [101, "John"];

6. Enum

Used for predefined constant values.

        enum Status {
           Active,
           Inactive,
           Pending
        }
        
        let currentStatus: Status = Status.Active;

7. Any

Can store any datatype.

        let value: any = 10;
        
        value = "Hello";
        value = true;

8. Unknown

Safer alternative to any.

        let data: unknown = "Hello";

9. Void

Used when a function does not return anything.

        function greet(): void {
           console.log("Hello");
        }

10. Null

        let value: null = null;

11. Undefined

        let value: undefined = undefined;

12. Object

Used for objects.

        let user: object = {
           name: "John",
           age: 30
        };

13. Union Type

Variable can hold multiple types.

        let id: string | number;
        
        id = 101;
        id = "EMP101";

14. Literal Types

Allows fixed values only.

        let direction: "left" | "right";
        
        direction = "left";

15. Never

Function never returns.

        function throwError(): never {
           throw new Error("Error");
        }

Playwright Questions

Q8. What is Playwright?

Playwright is an open-source end-to-end automation framework developed by Microsoft.

Supports Chromium, Firefox, WebKit and parallel execution.

Q9. Why Playwright over Selenium?

  • Faster execution
  • Auto-wait mechanism
  • Built-in API testing
  • Parallel execution
  • Better stability

Q10. What are Locators?

Locators are used to identify web elements.

page.locator('#username')

Q11. Best Locator Strategy?

  1. getByRole()
  2. getByLabel()
  3. getByText()
  4. data-testid
  5. CSS/XPath
