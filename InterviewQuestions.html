Q8. DATA types in Typescript

In TypeScript, data types define what kind of value a variable can hold.

1. Primitive types include string, number, boolean, bigint, symbol, null, and undefined.

Example: let name: string = "Alice";

Example: let age: number = 25;

Example: let isActive: boolean = true;

2. Object types are used for structured data like objects with properties.

Example: let user: { name: string; age: number } = { name: "John", age: 30 };

3. Array types store multiple values of the same type.

Example: let nums: number[] = [1, 2, 3];

4. Tuple types store fixed-length arrays with specific types.

Example: let person: [string, number] = ["Alice", 25];

5. Union types allow a variable to hold more than one type.

Example: let id: string | number = 101;

6. Literal types allow only specific exact values.

Example: let direction: "left" | "right" = "left";

7. Any type disables type checking and can hold any value.

Example: let data: any = 5;

8. Unknown type is similar to any but safer because it requires type checking before use.

9. Never type represents values that never occur, such as functions that always throw errors.

10. Void type is used for functions that do not return a value.

These types help TypeScript provide strong type safety and better code quality.

Playwright Questions

Q8. What is Playwright?

Playwright is an open-source end-to-end automation framework developed by Microsoft.

Supports Chromium, Firefox, WebKit and parallel execution.

Q9. Why Playwright over Selenium?

  • Faster execution
  • Auto-wait mechanism
  • Built-in API testing
  • Parallel execution
  • Better stability
