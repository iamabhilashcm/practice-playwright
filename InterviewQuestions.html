Manual Testing Questions

Q1. Difference between Severity and Priority?

Severity: Impact of the defect on functionality.

Priority: Urgency to fix the defect.

  • Login button not working → High Severity + High Priority
  • Typo in homepage → Low Severity + High Priority

Q2. What is STLC?

  2. Test Planning
  3. Test Case Development
  4. Environment Setup

Q4. Difference between == and === ?

== compares values only.

=== compares value + datatype.

5 == "5"   // true
5 === "5"  // false

Q5. What are Promises?

Promises handle asynchronous operations.

  • Pending
  • Resolved
  • Rejected

Playwright Questions

Q8. What is Playwright?

Playwright is an open-source end-to-end automation framework developed by Microsoft.

Supports Chromium, Firefox, WebKit and parallel execution.

Q9. Why Playwright over Selenium?

  • Faster execution
  • Auto-wait mechanism
  • Built-in API testing
  • Parallel execution
  • Better stability

Q10. What are Locators?

Locators are used to identify web elements.

page.locator('#username')

Q11. Best Locator Strategy?

  1. getByRole()
  2. getByLabel()
  3. getByText()
  4. data-testid
  5. CSS/XPath
page.getByRole('button', { name: 'Login' })
