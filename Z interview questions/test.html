TypeScript Data Types

In TypeScript, data types define what kind of value a variable can hold.

1. Primitive Types

Primitive types include: string, number, boolean, bigint, symbol, null, and undefined. 


let name: string = "Alice";
let age: number = 25;
let isActive: boolean = true;

2. Object Types

Object types are used for structured data with properties. 


let user: { name: string; age: number } = {
    name: "John",
    age: 30
};

3. Array Types

Array types store multiple values of the same type. 


let nums: number[] = [1, 2, 3];

4. Tuple Types

Tuple types store fixed-length arrays with specific types. 


let person: [string, number] = ["Alice", 25];

5. Union Types

Union types allow a variable to hold more than one type. 


let id: string | number = 101;

6. Literal Types

Literal types allow only specific exact values. 


let direction: "left" | "right" = "left";

7. Any Type

The any type disables type checking and can hold any value. 


let data: any = 5;

8. Unknown Type

The unknown type is similar to any but safer because it requires type checking before use. 


let value: unknown = "Hello";

if (typeof value === "string") {
    console.log(value.toUpperCase());
}

9. Never Type

The never type represents values that never occur. 


function throwError(message: string): never {
    throw new Error(message);
}

10. Void Type

The void type is used for functions that do not return a value. 


function greet(): void {
    console.log("Hello");
}

Conclusion

These data types help TypeScript provide strong type safety, better readability, and improved code quality.
